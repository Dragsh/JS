Задача 2. Создать массив объектов при помощи функции-конструктора Worker. В массиве должно быть не менее 10 экземпляров. В объекте Worker должны быть следующие поля:

  • фамилия и инициалы работника;
  • название занимаемой должности;
  • год поступления на работу;
  • имя файла с фотографией работника;
  • величина оклада работника;
  • метод формирования строки с данными работника: toString();
  • метод вычисления стажа работника для текущей даты.

Хранить массив в локальном хранилище, в формате JSON. Выполнить для массива работников (объектов Worker) при помощи кликов по экранным кнопкам или по выбранным клавишам следующие обработки:

  • вывод массива в исходном порядке;
  • вывод массива с упорядочиванием фамилий по алфавиту;
  • вывод массива с выделением работников с окладами, равными минимальному, упорядочивание по алфавиту;
  • вывод массива с выделением работников с окладами, равными максимальному, упорядочивание по должностям;
  • вывод массива с выделением работников с превышением заданного в строке ввода стажа работы, упорядочивать массив по окладу.

Разработайте форму (на той же странице, на которой выводится массив работников) для добавления работника, должность работника выбирайте из выпадающего списка. Все поля обязательные для заполнения. По кнопке «Отмена» все поля формы очищаются, по кнопке «Добавить» создается новый объект Worker, добавляется в массив объектов, выводится на страницу, массив обновляется в локальном хранилище.

Задание выводите в область, разворачиваемую по клику (используйте slideToggle() или подобные методы).

Реализуйте редактирование (в той же форме ввода) и удаление записей в массиве (кнопки редактирования и удаления добавляйте в коде).



*для увольнения нажмите левой кнопкой мыши на строку с данными сотрудника, которого хотите уволить и подтвердите действие
